## Web3Auth Vue Example

- ### Pre requirements:-

-  Install yarn:- npm i -g yarn

- Install yalc:- npm i -g yalc

- Build packages:-

    - Go to project root folder: cd ../../
    - Build project:- yarn run local-publish

- Run the example:-

    - Go to example: cd example/vue-app
    - Run example:- yarn run serve:local
